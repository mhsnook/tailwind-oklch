OKLCH Composer

A color utility system for Tailwind v4 that exposes luminance, chroma, and hue as independent, composable axes in your class names. No more opacity hacks. Re-theme your entire app by changing one hue variable.

Theme Hues

5×5 Luminance × Chroma Grid

Each matrix shows all combinations of luminance (rows) and chroma (columns) for a given hue. Hover to see the class name.
